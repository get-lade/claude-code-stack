### The hypothesis
`agents/*.md` declared tools as a lowercase YAML list (`- read`, `- web_search`).
Claude Code tool names are **PascalCase** (`Read`, `WebSearch`). Lowercase matches
nothing → agent gets an **empty toolset** → it can't emit real `tool_use`, so it
writes `<invoke name="Read">…` as literal text and hallucinates results. That is
exactly the F2 symptom seen last session.

Confidence: ~95% it's *a* bug; ~70% it's *the* F2 cause; ~65% this fix resolves it.
